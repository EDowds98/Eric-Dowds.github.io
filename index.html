Projects

  • Discord Bot Written in Java


    I developed a Discord bot that read in messages, compared each word of the message to a textfile of keywords, and returned a message back depending on whether a keyword was found. This was my first time working with an API, and although I struggled at time I managed to push through and deliver a working bot. The most exciting part for me is how extensible the project is - I'm looking into hosting the bot on my raspberry pi, using more efficient data structures, and adding commands to give users more control over it. Source code is available at the GitHub repository.

  • AVR Piezo Musical Instrument


    I built a simple musical instrument that would play notes from a buzzer depending on which button was pressed. I just did it for fun as I had a few pieces lying around. I really enjoyed programming an actual project outside of university, and I learned how to add my own header files and how to build files in the terminal as I didn't use an IDE. You can see more details at its repository.

  • A Simple Stack in C


    I was enjoying learning about Data Structures and Algorithms in University so I thought I'd implement a simple Stack. I am using it as an opportunity to practise using structs and pointers to functions in C to program a little more OOPly. I'm finding that it helps me appreciate OOP programming a bit more, although I do how elegant a solution pointers are to problems. Once again the details are in its repository
